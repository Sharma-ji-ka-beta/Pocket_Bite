# PocketBite

Turn surplus food into savings.

PocketBite is a full-stack platform that helps reduce food waste by enabling bakeries to list surplus items with defined shelf life. Users can browse real-time listings, add items to cart, and place orders before they expire.

⚙️ Features
Login (lightweight)
Dynamic listings (FastAPI + Supabase)
Expiry-based filtering
Cart system (localStorage)
Payment simulation (COD / Card)
Auto cart reset after checkout
🏗️ Tech Stack
Frontend: HTML, Tailwind, JavaScript
Backend: FastAPI
Database: Supabase
